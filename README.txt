# Optimizations for Multishop Template

This repository contains the Multishop template with various optimizations applied.

## Optimizations Implemented

- Added Vite for faster development and bundling.
- Minified CSS and JavaScript assets.
- Used CDN links for common libraries.
- Implemented lazy loading for images.
- Marked scripts as async or defer.
- Added preloading hints for critical CSS.

## How to Run

1. Clone this repository.
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Build for production: `npm run build`
